hqc-py
======

[![Python CI](https://github.com/yodalee/hqc-py/actions/workflows/python-ci.yml/badge.svg?branch=master)](https://github.com/yodalee/hqc-py/actions/workflows/python-ci.yml)

A pure python implementation of Hamming Quasi-Cyclic (future FIPS 207)

> [!CAUTION]
> :warning: **Under no circumstances should this be used for cryptographic
applications.** :warning:
>
> This is an educational resource and has not been designed to be secure
> against any form of side-channel attack. The intended use of this project
> is for learning and experimenting with ML-DSA and Dilithium

This repository contains a pure python implementation of **HQC**, the NIST
Code-Based Key Encapsulation Mechanism (KEM). Follow the [current spec](https://pqc-hqc.org/doc/hqc_specifications_2025_08_22.pdf) submitted to NIST post-quantum project.
